## What is this template?

Text-to-speech and zero-shot voice cloning in your browser, powered by
[Chatterbox](https://github.com/resemble-ai/chatterbox) (Resemble AI). Type text, pick or
clone a voice from a few seconds of reference audio, and generate natural speech — with a
web UI **and** an OpenAI-compatible `/v1/audio/speech` API. Includes the fast Turbo model
(runs on ~4 GB VRAM) plus multilingual voices. Every clip carries an imperceptible Perth
watermark by design.
